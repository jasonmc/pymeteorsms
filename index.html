Pymeteorsms

Python program to send text messages via meteor website. Aims to be similar to classic meteorsms Perl script.

Download as .zip Download as .tar.gz View on GitHub

PyMeteorSMS

Unfortunately meteor changed their web interface recently, which broke the classic meteorsms Perl script. Initially I looked at fixing that script, but it uses TestGen4Web which is practically dead now. I decided to reverse engineer the site myself. They now use ajax requests for lots of things, which would've made TestGen4Web useless anyhow. It does provide us with essentially an API however. I first used mechanize, but due to some bug with https and proxies, I couldn't use it from one of my computers, so I created a version using just urllib2. This has the added bonus of making substantially less requests and therefore is faster.

This code supports the original meteorsms config script syntax.

Dependencies

Python 2.6 onwards include required libraries.

Install

Just run the script :)

$ bin/meteorsms.py mark
[ recipient : mark (+35387xxxxxxx) ]
